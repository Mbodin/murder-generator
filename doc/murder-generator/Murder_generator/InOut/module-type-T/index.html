Create a text input meant to return a list of things. Each time that the user type a string, it is fed to its argument function. The strings of the returned list is shown to the user. If the user chooses one of these elements, it is added to a list displayed next to the input. This final list can be fetched with the function returned with the node. The initial list and a placeholder string is given to the function to help the user.
